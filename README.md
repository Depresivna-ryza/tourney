# Tourney
A desktop tournament manager built with C# .Net and Avalonia (MVVM).

https://github.com/user-attachments/assets/b95ecbcf-e8b8-4f0a-8a89-3f7191369888

### Features
  - Teams with descriptions, icons, colors
  - Create, Read, Update, Delete (CRUD) operations on teams and tournaments
  - Elimination tournament management:
    - dynamic size of brackets
    - automatic and asynchronous bracket progression
  - Versus (1 vs. 1) tournament management
  - Persistance of teams and tournaments
  - Visual history of all previous tournaments

### Prerequisites

`.NET SDK` (8.0 or later)

### Running the application

```
dotnet run
```
